How to Join DynastyMC

Play on Java or Bedrock Edition

Java Edition

1. Open Minecraft Java Edition.

2. Click Multiplayer → Add Server.

3. Enter the server address:

play.dynastymc.net

4. Join the server and start playing!

Bedrock Edition

1. Open Minecraft Bedrock Edition.

2. Go to Servers → Add Server.

3. Enter the details:

IP: play.dynastymc.net
Port: 19132

4. Save and join DynastyMC!

How to Register & Login

Secure your account on DynastyMC

Register

When you join for the first time, type this command in chat:

/register

Example:

/register mypassword mypassword

Login

Every time you rejoin the server, type:

/login

Example:

/login mypassword
